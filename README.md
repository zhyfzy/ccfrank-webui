# 中国计算机学会推荐国际学术会议和期刊目录 - 2022

这是一个基于Vue 3的美观Web界面，用于展示中国计算机学会推荐的国际学术会议和期刊目录（2022年版）。

🌐 **在线访问**：https://ccfrank.zhyfzy.com/

## 功能特点

- 📋 左侧目录导航，支持10个主要分类（默认展开所有二级目录）
- 📑 每个分类下包含"会议"和"期刊"两个子分类
- 🎯 右侧内容区域，支持A/B/C类快速切换和跳转
- 🔗 所有网址自动转换为可点击的超链接
- 📱 完整的移动端适配，窄屏时目录可收起/展开
- 🎨 现代化的UI设计，响应式布局
- 📌 标题栏始终固定在顶部，显示当前查看的分类信息

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用说明

### 桌面端
- 左侧目录默认展开，可直接点击分类和子分类
- 右侧显示内容，点击A/B/C类按钮可快速跳转到对应内容区域
- 标题栏固定在顶部，显示当前查看的分类信息

### 移动端
- 左侧目录默认收起，点击顶部菜单按钮（☰）可打开
- 点击遮罩层或关闭按钮可关闭目录
- 选择目录项后自动关闭侧边栏
- 所有内容自适应屏幕宽度

## 浏览器支持

支持所有现代浏览器（Chrome, Firefox, Safari, Edge等）和移动端浏览器
